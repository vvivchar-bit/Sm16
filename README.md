# Тема 16. Об’єктно-орієнтоване програмування

Дисципліна: **Основи програмної інженерії**  
Тип роботи: **Самостійне опрацювання**

## Опис
Репозиторій містить теоретичний звіт з теми
«Об’єктно-орієнтоване програмування», у якому розкрито
сутність ООП, основні принципи (інкапсуляція, наслідування,
поліморфізм, абстракція), переваги та недоліки об’єктно-
орієнтованого підходу, а також його роль у сучасній
програмній інженерії.

## Вміст репозиторію
- `Sm16.pdf` — оформлений звіт з теми
- `README.md` — опис репозиторію

## Примітка
Практична реалізація та програмний код не передбачені
умовою завдання, оскільки тема має теоретичний характер.
